# Azure User Provisioning Automation

PowerShell scripts for provisioning users between on-premises Active Directory and Azure AD. Includes automation for bulk user and group creation, CSV imports, and hybrid identity management.

## Project Structure

- `Scripts/Core/` - User and group provisioning scripts
- `Scripts/Setup/` - Initial configuration and setup
- `Scripts/Import-UsersFromCSV.ps1` - Bulk CSV import
- `Config/` - Configuration files and templates
- `Data/Templates/` - CSV templates

## Technologies

PowerShell 7, Windows Server 2022, Active Directory, Azure AD, Microsoft Graph API
